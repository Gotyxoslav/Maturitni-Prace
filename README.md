General information for the Toney project

  What do I need to run this project?
        
        Python (woah)
        PyQt5
        Mysql
        Argon2
        Flask
        Os
        
        os should be installed automatically with python, for the rest, use "pip install Flask argon2-cffi mysql-connector-python uuid pyqt5" in the command console

  How do I run this project?
        
        You simply go to "Maturitni-Prace\maturita\views\main.py" to run the website (do note, in order to add/edit/delete music and albums, you need to have the admin or creator role), and "Maturitni-Prace\maturita\views\adminapp.py" to run the administration app (do note, to log in the administration app, you need to have the admin role)
